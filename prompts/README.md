
# Salary Management System - Enterprise SPDD

## Stack
- Frontend: React.js + Vite + Material UI
- Backend: FastAPI
- Database: SQLite
- ORM: SQLAlchemy
- Authentication: JWT
- Testing: Pytest + React Testing Library
- Architecture: MVC + Service + Repository Pattern

## Core Features
- JWT Authentication
- Employee CRUD
- Salary Analytics
- Country Insights
- Dashboard Metrics
- Bulk Seeding for 10,000 employees
- Unit & Integration Tests
- TDD Workflow
